Overview

A utility class that encapsulates a specific request with its parameters. Users will rarely create a Request directly; you should instead use the #call method on the relevant Service object.

def initialize(service, method, params)
  @service, @method, @params = service, method, params
  @start_time = Time.now
  @attempts = 0
  
  self.callback {|r| @response = r}
  self.errback {|r| @response = r}
end

  end
end
